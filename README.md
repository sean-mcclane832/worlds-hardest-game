# World's Hardest Game

Run this project locally to play in any modern browser.

## Quick start

1. Install dependencies:

   ```sh
   npm install
   ```

2. Start the web server:

   ```sh
   npm start
   ```

3. Open the game in your browser:

   ```text
   http://localhost:3000
   ```

If you are deploying to a host, set the `PORT` environment variable to the port your platform expects.
